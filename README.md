## 🛠 Built With

My personal portfolio website, featuring my GitHub projects, resume, and technical skills.

Built using:

- React.js  
- Node.js  
- Express.js  
- MongoDB  
- CSS3 / Tailwind / Bootstrap  
- VSCode  
- Vercel

## ✨ Features

**📖 Multi-page layout**  
**🎨 Styled with React-Bootstrap and custom CSS**  
**📱 Fully responsive on all devices**
